### VS 코드에서(관리자 권한으로 실행)
1. python -V (파이썬 버전 확인)
2. virtualenv 확인
3. 없으면 설치 pip install virtualenv
4. virtualenv venv 입력
5. Set-ExecutionPolicy Unrestricted 입력
6. .\venv\Scripts\activate
7. pip install pandas bs4
8. pip install selenium webdriver_manager
